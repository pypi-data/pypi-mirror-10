# edc-base

Base model, manager, field, validator, form and admin classes for Edc. 


Installation
------------

	pip install edc-base

In the settings file add:

	STUDY_OPEN_DATETIME = datetime.today()
	STUDY_CLOSE_DATETIME = datetime.today()
